# 🧪 Greater than - Less than challenge (28/11/2023)

```javascript
console.log(1 < 2 < 3);

console.log(3 > 2 > 1);
```

### ✅ Solution
The expected output is:
- True
- False

### 📝 Explanation
The first one is `true` because `1 < 2 = true` and `true == 1` so `1 < 3 = true`.

The second one is `false` because `3 > 2 = true` and `true == 1` so `1 < 1 = false`.
