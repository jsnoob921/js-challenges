# 🧪 Find the X value challenge (I) (18/10/2023)

```javascript
const x = ??;

if (x !== x) {
    console.log('Hello World');
}
```

### ✅ Solution
The value of X must be `NaN`.

### 📝 Explanation
- If Type(x) is Number, then
    - If x is NaN, return false.
    - If y is NaN, return false.
