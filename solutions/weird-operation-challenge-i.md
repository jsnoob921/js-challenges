# 🧪 Weird operation challenge (I) (30/11/2023)

```javascript
console.log(+!![]);
```

### ✅ Solution
The expected output is: `1`.

### 📝 Explanation
The expression `!!![]` returns a truthy value, adding the `+` symbol brings type coercion in JavaScript into play and turns `true` into `1`.
